# 🔐 Password Generator

A simple and responsive Password Generator built using HTML, CSS, and JavaScript.

## 🚀 Features

* Generate strong and secure passwords
* Adjustable password length using slider
* Include:

  * Uppercase Letters
  * Lowercase Letters
  * Numbers
  * Symbols
* Password Strength Indicator
* One-click Copy Password feature
* Responsive and modern UI

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Use

1. Select the desired password length.
2. Choose the character types you want.
3. Click **Generate Password**.
4. Copy the generated password using the copy icon.
